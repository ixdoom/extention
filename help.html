Инструкция по применению

  1. Для перехода в защищенный режим диалога перейдите на нужную вам страницу диалога и нажмите на символ открытого замка . В защищенном режиме все исходящие и входящие сообщения будут зашифрованы и расшифрованы соответственно указанным ключом.
  2. Для установки ключа шифрования нажмите на символ ключа , после чего введите требуемый ключ и нажмите на кнопку ОК
  3. Для отправки сообщения введите в окно сообщения в защищенном режиме текст сообщения и нажмите на клавишу Enter.
  4. Для выхода из защищенного режима нажмите на символ замка

О расширении

Данное расширение предназначено для шифрования личных сообщений социальной сети «ВКонтакте». Для защиты переписки следует использовать определенную при личной встрече или иным способом ключевую фразу (пароль). После ввода в поле пароля ключевой фразы все текстовые сообщения, отправляемые из защищенного режима, будут зашифрованы: прочесть их можно с помощью данного расширения и известной ключевой фразы.

Общие сведения о защите информации

Данное расширение обеспечивает защиту от прочтения сообщений со стороны любой группы лиц, получившей доступ к переписке. Однако при несоблюдении определенных правил содержимое переписки все равно может стать известным злоумышленнику. Перечисленные далее советы помогут избежать случаев возможного разглашения информации.

  • Храните в секрете ключевую фразу. В большинстве случаев злоумышленник получает доступ к защищенной информации посредством раскрытия ключевой фразы. Постарайтесь запомнить её, не записывая на сторонние носители информации: бумага, телефон, флеш-карта и др. Не произносите её вслух без необходимости.
  • Не используйте открытые каналы связи для передачи ключевой фразы. Наиболее приемлемым способом передачи пароля для шифрования сообщений является личная встреча участников переписки. В иных случаях постарайтесь определить пароль исходя из сведений, известным только участникам переписки.
  • Убедитесь в отсутствии вредоносных программ на используемой машине. Кроме того, следует осторожно относиться к использованию чужих компьютеров для передачи сообщений. Существуют средства, способные сохранять вводимые с клавиатуры символы и обеспечивающие их передачу злоумышленнику. Это может повлечь за собой раскрытие данных учетной записи, пароля для переписки, содержимого самой переписки.
  • Не используйте простые ключевые фразы. Кроме того, не используйте в качестве ключевых фраз какой-либо из используемых одним из участников переписки паролей. Простая ключевая фраза может быть достаточно быстро подобрана злоумышленником, что приведет к раскрытию данных.

Особенности работы расширения

Ключевая фраза, закрепленная за конкретным диалогом, запоминается расширением до момента закрытия вкладки «ВКонтакте» или выхода из учетной записи. Переписка может одновременно поддерживаться во множестве диалогов, для каждого из них будет запомнен отдельный пароль. Ключевой фразой по умолчанию является пустая строка. Максимальная длина ключевой фразы ограничена.

Признаком корректно расшифрованного сообщения является светло-зеленый фон текста сообщения. Если цвет фона принятого сообщения бордовый, то принятое сообщение было распознано расширением как недействительное. Оно, скорее всего, является повторно отправленной копией одного из предыдущих сообщений, что может быть выполнено злоумышленником.

Максимальный размер отправляемого сообщения ограничен. Для сообщения, состоящего целиком из символов латинского алфавита, десятичных цифр, знаков препинания и специальных символов, этот размер составляет 2500 символов. Для сообщений на русском языке этот объем в среднем составляет 1200 символов.

Пересылка сообщения путем копирования его содержимого и повторной отправки не приведет к успешному результату: сообщение либо не будет расшифровано, либо будет распознано как недействительное.

Особенности реализации расширения

Расширение использует текущее локальное время компьютера в качестве одного из параметров шифрования. Изменение локального времени повлечет за собой неопределенное поведение расширения.

Расширение не поддерживает работу с выносимыми окнами диалогов.

Расширение не поддерживает работу с мобильными приложениями. Таким образом, и отправление зашифрованных сообщений, и расшифровка входящих зашифрованных сообщений с мобильных устройств без помощи сторонних средств невозможна.

Расширение не поддерживает защиту прикрепляемых к сообщениям документов: шифруется только текст.

В качестве алгоритма шифрования используется алгоритм AES. Для защиты от подбора паролей по словарю на основе ключевых фраз пользователей генерируется криптографический ключ.
